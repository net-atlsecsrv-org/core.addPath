# PI (Azure Cosmos DB)
 Returns the constant value of PI.  
  
## Syntax
  
```sql
PI ()  
```  
   
## Return types
  
  Returns a numeric expression.  
  
## Examples
  
  The following example returns the value of `PI`.  
  
```sql
SELECT PI() AS pi 
```  
  
 Here is the result set.  
  
```json
[{"pi": 3.1415926535897931}]  
```
