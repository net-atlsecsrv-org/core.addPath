Now that you've learned the basics of Azure Table storage, follow these links to learn about more complex storage tasks.

* View the Table Service reference documentation in the [Azure Storage Client Library for .NET](/dotnet/api/overview/azure/storage) reference for complete details about available APIs.
* To learn how to simplify the code you write to work with Azure Storage, check out [What is the Azure WebJobs SDK](https://github.com/Azure/azure-webjobs-sdk/wiki)
* View more feature guides to learn about additional options for storing data in Azure.
  * To work with Azure Storage Blobs, see [Get Started with Azure Blob storage using .NET](../articles/storage/blobs/storage-quickstart-blobs-dotnet.md).
  * To work with Azure Storage Queues, [Get started with Azure Queue storage using .NET](../articles/storage/queues/storage-dotnet-how-to-use-queues.md).
  * To store relational data, see [Connect to SQL Database by using .NET (C#)](../articles/azure-sql/database/connect-query-dotnet-visual-studio.md).
