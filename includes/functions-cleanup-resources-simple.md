## Clean up resources

After you have finished working with the resources created by this article, use the following command to delete them from your subscription:

```azurecli-interactive
az group delete --name myResourceGroup
```

Type `y` when prompted. This command removes the `myResourceGroup` resource group and all of the resources that belong to it.
