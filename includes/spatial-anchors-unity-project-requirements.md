The Azure Spatial Anchors SDK for version 2.9 or later should be configured in [Unity 2020.3 (LTS)](https://unity3d.com/unity/whats-new/2020.3.0) with the [Unity XR Plug-in Framework](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) using the following packages:

- AR Foundation: [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@4.0/manual/index.html)
- Windows XR Plugin (for HoloLens device support) : [4.4.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr@4.4/manual/index.html)
- ARCore XR Plugin (for Android device support): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.0/manual/index.html)
- ARKit XR Plugin (for iOS device support): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arkit@4.0/manual/index.html)

The Azure Spatial Anchors SDK versions 2.8 or earlier do not support the [Unity XR Plug-in Framework](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) and should be configured in [Unity 2019.4 (LTS)](https://unity.com/releases/2019-lts) using the following packages:

- AR Foundation: [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html)
- Windows Mixed Reality (for HoloLens device support): [4.2.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html)
- ARCore XR Plugin (for Android device support): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html)
- ARKit XR Plugin (for iOS device support): [3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html)
