The Azure Spatial Anchors SDK should be configured in <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019.4 (LTS)</a> using the following packages:
- AR Foundation: <a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality: <a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- ARCore XR Plugin (for Android device support): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR Plugin (for iOS device support): <a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> The ASA SDK does not currently support the <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">Unity XR plug-in framework</a>
