The entries in the `deploymentconfig.json` document map to the parameters for [LocalWebservice.deploy_configuration](/python/api/azureml-core/azureml.core.webservice.local.localwebservicedeploymentconfiguration?view=azure-ml-py). The following table describes the mapping between the entities in the JSON document and the parameters for the method:

| JSON entity | Method parameter | Description |
| ----- | ----- | ----- |
| `computeType` | NA | The compute target. For local targets, the value must be `local`. |
| `port` | `port` | The local port on which to expose the service's HTTP endpoint. |

This JSON is an example deployment configuration for use with the CLI:

```json
{
    "computeType": "local",
    "port": 32267
}
```
