# Virtual machines documentation
Azure Virtual Machines provide an on-demand, high-scale, and secure virtualized infrastructure. You can create Windows virtual machines or Linux virtual machines that run Red Hat, Ubuntu, or the Linux distribution of your choice. To learn how, see:

- [Create Windows virtual machine](/azure/virtual-machines/windows/quick-create-portal/?WT.mc_id=UIHelpPilot)
- [Create Linux virtual machine](/azure/virtual-machines/linux/quick-create-portal/?WT.mc_id=UIHelpPilot)
